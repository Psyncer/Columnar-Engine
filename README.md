# Columnar-Engine

Реализация колоночного движка с нуля.

## Структура проекта

- `src/execution` - операторы и выражения.
- `src/storage` - основные структуры данных.
- `src/io` - чтение и запись данных.
- `src/runner` - построение и запуск запросов.
- `tools` - вспомогательные утилиты.
- `script` - скрипты для сборки, конвертации и запуска запросов.
